# Diabetes Prediction Model

## Overview
This project predicts diabetes using machine learning classification techniques.

## Technologies Used
- Python
- Scikit-learn

## Machine Learning Model
- Support Vector Machine (SVM)

## Workflow
1. Data preprocessing
2. Feature scaling
3. Model training
4. Cross-validation
5. Performance evaluation

## Evaluation Metrics
- Accuracy
- ROC-AUC Score
- Cross-validation

## Results
- Mean Cross-validation Accuracy: 77.2%
- ROC-AUC Score: 0.70

## Dataset
Diabetes dataset containing medical diagnostic features.
